# Guestbook

Guestbook is a travel discovery platform connecting Airbnb guests with local towns.

## Features

- Airbnb discovery
- Local business discovery
- Interactive maps
- Reviews
- QR code guest reviews
- Local recommendations

## Tech Stack

Frontend:
- React Native
- Expo

Backend:
- Supabase

Database:
- PostgreSQL
